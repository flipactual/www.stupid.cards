## Scripts

### `build` – build the app

```sh
npm run build
```

### `start` – run a server for the app

```sh
npm run start
```

### `test` – run the tests

```sh
npm run test
```

### `coverage` – generate and view code coverage as HTML

```sh
npm run coverage
```

### `lint` – lint the codebase

```sh
npm run lint
```

### `readme` – generate the README

```sh
npm run readme
```
